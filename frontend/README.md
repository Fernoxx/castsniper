# Frontend Web App

Next.js frontend for managing the Farcaster Creator Coin Sniper.

## Features

- Real-time status monitoring
- Add/remove users to monitor
- View monitored users
- Manual check trigger
- Responsive design

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Production

```bash
npm run build
npm start
```

## Environment Variables

- `NEXT_PUBLIC_API_URL` - Your Railway backend URL (required)

## Deployment

Deploy to Vercel - see `DEPLOYMENT.md` for details.
